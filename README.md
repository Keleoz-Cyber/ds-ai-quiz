# 数据结构智能刷题与错题推荐系统

## 项目简介

这是一个基于C++开发的数据结构智能刷题系统，集成了错题本管理和AI智能推荐功能。系统通过分析用户的做题记录，智能推荐最需要练习的题目，帮助用户高效复习数据结构知识点。

## 功能特性

- **随机刷题模式**：从题库中随机抽取题目进行练习
- **错题本练习**：专门针对答错的题目进行强化练习
- **AI智能推荐**：基于多维度算法智能推荐最适合当前练习的题目
- **模拟考试模式**：支持自定义题目数量的模拟考试，提供详细成绩报告
- **知识点复习路径推荐**：基于知识点依赖图，智能生成从基础到目标的复习路径
- **做题统计分析**：提供总体统计和按知识点分类的详细统计数据
- **持久化存储**：自动保存做题记录，支持历史数据加载

## AI推荐算法

系统采用多维度评分机制进行题目推荐：

1. **错误率权重（60%）**：优先推荐错误率高的题目
2. **时间间隔权重（30%）**：推荐长期未练习的题目
3. **难度权重（10%）**：考虑题目难度（1-5级）
4. **未做题奖励**：为从未做过的题目增加额外推荐分数

## 数据格式

### 题库文件格式 (data/questions.csv)

每行一道题，字段用竖线 `|` 分隔：

```
题号|题目描述|选项A|选项B|选项C|选项D|正确答案索引|知识点|难度等级
```

示例：
```
1|数组的随机访问时间复杂度是？|O(1)|O(n)|O(log n)|O(n^2)|0|数组|1
2|二叉树的先序遍历顺序是？|根左右|左根右|左右根|根右左|0|树|2
```

### 做题记录格式 (data/records.csv)

自动生成，每行一条记录：

```
题号|是否正确(0/1)|用时(秒)|时间戳
```

### 知识点依赖图格式 (data/knowledge_graph.txt)

定义知识点之间的前置关系，每行一个知识点：

```
当前知识点|前置知识点1,前置知识点2,...
```

示例：
```
线性表|
栈|线性表
队列|线性表
树与二叉树|栈,队列
图|树与二叉树
```

说明：
- 如果某知识点没有前置（如"线性表"），竖线后留空
- 多个前置知识点用逗号分隔
- 该文件用于生成复习路径推荐

## 编译与运行

### 环境要求

- C++11 或更高版本
- CMake 3.0 或更高版本
- Windows/Linux/MacOS

### 编译步骤

```bash
# 创建构建目录
mkdir build
cd build

# 使用CMake生成构建文件
cmake ..

# 编译
cmake --build .
```

### 运行程序

```bash
# 确保data目录存在且包含questions.csv
./DS_AI_Quiz
```

## 使用说明

1. **首次使用**：
   - 在 `data/` 目录下创建 `questions.csv` 文件
   - 按照格式添加题目数据

2. **主菜单选项**：
   - `1` - 随机刷题：随机抽取一道题目练习
   - `2` - 错题本练习：从错题集中随机抽取题目
   - `3` - AI智能推荐：推荐5道最适合练习的题目
   - `4` - 做题统计查看：查看总体统计和知识点统计
   - `5` - 模拟考试模式：进行自定义题量的模拟考试
   - `6` - 知识点复习路径推荐：基于依赖关系规划复习路径
   - `0` - 退出程序

3. **答题方式**：
   - 输入选项编号（0-3）提交答案
   - 系统自动记录答题时间和正确性

## 核心数据结构

### Question（题目结构）
```cpp
struct Question {
    int id;                    // 题目编号
    string text;               // 题目描述
    vector<string> options;    // 选项列表
    int answer;                // 正确答案索引
    string knowledge;          // 知识点
    int difficulty;            // 难度（1-5）
};
```

### Record（做题记录）
```cpp
struct Record {
    int questionId;      // 题号
    bool correct;        // 是否答对
    int usedSeconds;     // 作答用时
    long long timestamp; // 时间戳
};
```

### QuestionStat（题目统计）
```cpp
struct QuestionStat {
    int totalAttempts;       // 总作答次数
    int correctAttempts;     // 答对次数
    int totalTime;           // 累计作答时间
    long long lastTimestamp; // 最近一次作答时间
};
```

## 模拟考试模式

模拟考试模式提供完整的考试体验：

### 功能特点
- **自定义题量**：用户可自由设定考试题目数量（1 ~ 题库总数）
- **随机抽题**：使用现代随机算法从题库中随机抽取互不重复的题目
- **即时反馈**：每道题作答后立即显示正确答案
- **详细报告**：考试结束后生成包含以下内容的成绩报告：
  - 总题数、答对题数、答错题数
  - 本次考试正确率
  - 按知识点分类的作答统计（题数、正确数、正确率）
- **记录保存**：考试记录自动保存到历史记录中

### 使用流程
1. 在主菜单选择"5. 模拟考试模式"
2. 输入本次考试的题目数量
3. 依次完成所有题目
4. 查看系统生成的考试成绩报告

## 知识点复习路径推荐

基于知识点依赖图的智能复习路径规划功能：

### 核心原理
- **依赖图建模**：使用邻接表存储知识点之间的前置依赖关系
- **拓扑排序**：通过DFS深度优先搜索生成从基础到目标的学习路径
- **掌握度分析**：结合历史做题记录，评估每个知识点的掌握情况

### 功能特点
- **可视化展示**：按正确率从低到高排列所有知识点
- **智能路径生成**：自动分析目标知识点的所有前置依赖
- **薄弱环节标注**：在路径中突出显示掌握不足的知识点（正确率<60%）
- **未练习提示**：标注尚未练习过的知识点

### 使用流程
1. 在主菜单选择"6. 知识点复习路径推荐"
2. 查看当前所有知识点的掌握情况（按掌握度排序）
3. 选择想要复习的目标知识点编号
4. 系统生成完整的复习路径，从基础前置到目标知识点
5. 根据推荐路径安排学习计划

### 推荐算法
```
对于目标知识点 T：
1. 递归查找 T 的所有前置知识点
2. 使用 DFS 遍历依赖树
3. 按拓扑顺序组织学习路径：前置知识点 → 目标知识点
4. 标注每个节点的掌握情况（题数、正确率）
5. 突出显示薄弱环节供重点复习
```

## 技术特点

- **图数据结构**：使用邻接表存储知识点依赖关系
- **哈希表**：unordered_map 实现 O(1) 快速查询
- **优先队列**：priority_queue 实现高效推荐排序
- **DFS算法**：深度优先搜索生成拓扑排序的复习路径
- **现代随机算法**：std::shuffle + mt19937 保证抽题随机性
- **文件流操作**：实现数据持久化
- **精确计时**：chrono库实现毫秒级计时
- **跨平台支持**：Windows平台支持UTF-8中文显示

## 核心算法

### AI推荐算法
基于多维度评分：错误率(60%) + 时间间隔(30%) + 难度(10%) + 未做奖励

### 知识图拓扑排序
使用DFS深度优先遍历依赖树，生成从基础到高级的学习路径
